<h3>A react native app with basic firebase auth</h3>

Insert your own google-services.json file from firebase in /android/app folder

To run:
----------------
<p>npm install</p>
<p>npm start</p>

<p>and finally:</p>
<p>npm run android</p>

<p>or whatever it is for IOS</p>

** Not tested in IOS **

Based on react-native-firebase-starter kit
